###ch03: 
    JS的数据类型分为两类：原始数据类型和对象类型。
    原始类型包括：数字、字符串和布尔值。
    两个特殊的原始值：null 和 undefined
    除了那5个就是对象了，其中包括：数组类、函数类、日期类、正则类和错误类。
    
    原始值(undefined、null、布尔值、数字和字符串)是不可改变的；
    对象(数组和函数)是可以改变的。
